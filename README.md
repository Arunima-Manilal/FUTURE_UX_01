Glow & Grace Salon Website Redesign
Project Overview

This project is a UI/UX redesign of a salon website created for the Future Interns UI/UX Design Internship Task 1. The website focuses on improving lead generation, appointment bookings, user engagement, and conversion rates through a clean and modern design.
